UPGRADE NOTES:

If you upgrade from v 6.0 or older, you can not keep your playlists & favourites. Sorry for the inconvenience.
If you upgrade from v 7.x using the deb package, playlists & starred items will be kept.

HOW TO UPGRADE:

http://code.google.com/p/spotcommander/wiki/Upgrade

CHANGELOG:

 * Some UI changes
 * Support for browser history (back/forward)
 * Optimizations for the new version (v 1.2) of the Android app (see Google Play)
 * Swipe in from the right to go back when running fullscreen on iOS
 * Remember where you left off when running fullscreen on iOS
 * When using the Android app, if the app is killed by the system, remember where you left off
 * Minor changes & bug fixes
